# Kindergarten-cz-pack
Školka s českou fanouškovskou lokalizací

Kindergarten with Czech fan-made localization

(Samozřejmě měj zakoupenou hru prvně)
(Of course, have the base game purchased first)

Jak na instalaci:
1) Rozbal .zip (ujisti se, že máš zálohu hry, kdyby se něco pokazilo)
2) Zkopíruj soubory do složky Kindergarten (měla by být v: C:\Program Files (x86)\Steam\steamapps\common\Kindergarten\Kindergarten_Data), a to do složky "Kindergarten_Data" i "Managed"
   2.1) Stejný postup je i pro KG2/3 apod.
3) Spusť hru!

How to install:
1) Extract the .zip file (make sure you have a backup of the game if something goes wrong, always better to be safe than sorry)
2) Copy files into your Kindergarten folder (should be in: C:\Program Files (x86)\Steam\steamapps\common\Kindergarten\Kindergarten_Data), both "Kindergarten_Data" and "Managed" folder
   2.1) Same path is for KG2/3 etc.
3) Launch the game!
